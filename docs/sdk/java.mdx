An easy-to-use library designed to simplify interaction with the api.megoru.ru

[GitHub Repo](https://github.com/megoRU/api.megoru.ru-wrapper)

# Installation {#install}

Check the latest version: [jitpack.io](https://jitpack.io/#megoRU/api.megoru.ru-wrapper)

Add to your pom.xml file:

```maven
<repositories>
 <repository>
	<id>jitpack.io</id>
	<url>https://jitpack.io</url>
 </repository>
</repositories>

<dependency>
    <groupId>com.github.megoRU</groupId>
    <artifactId>api.megoru.ru</artifactId>
    <version>5.0.5</version>
</dependency>
```

# Basic Usage {#usage}

```java
public static void main(String[] args) {
    MegoruAPI megoruAPI = new MegoruAPI.Builder().build();
    String word = megoruAPI.getWord(gameWordLanguage).word();

    System.out.println(word);
}
```
