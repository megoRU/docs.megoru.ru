Простая и удобная библиотека для взаимодействия с [api.megoru.ru](https://api.megoru.ru).

[GitHub репозиторий](https://github.com/megoRU/api.megoru.ru-wrapper)

# Базовое использование{#usage}

```Java
public static void main(String[] args) {
    MegoruAPI megoruAPI = new MegoruAPI.Builder().build();
    String word = megoruAPI.getWord(gameWordLanguage).word();

    System.out.println(word);
}
```
