# Comment — определение содержимого сообщения {#comment}

:::info Base API URL
`https://api.megoru.ru`
:::

## Конечная точка

| Метод | Endpoint       | Описание                       |
|-------|----------------|--------------------------------|
| POST  | `/api/comment` | Определяет категорию сообщения |

---

## Тело запроса (Request Body)

| Параметр  | Тип    | Обязательно | Описание                    |
|-----------|--------|-------------|-----------------------------|
| `comment` | String | Да          | Текст сообщения для анализа |

### Пример запроса

```json
{
  "comment": "Купи KVN со скидкой 90% https://example.com"
}
```

---

## Заголовки

| Имя            | Тип    | Описание           |
|----------------|--------|--------------------|
| `Content-Type` | String | `application/json` |

---

## Ответ сервера

### 200 OK — Сообщение успешно обработано

| Поле     | Тип    | Описание                         |
|----------|--------|----------------------------------|
| `result` | String | Определённая категория сообщения |

#### Пример ответа

```json
{
  "result": "SPAM"
}
```

---

## Возможные значения `result`

| Значение        | Описание                  |
|-----------------|---------------------------|
| `NORMAL`        | Обычное сообщение         |
| `SPAM`          | Спам                      |
| `ADVERTISEMENT` | Реклама                   |
| `PORNOGRAPHY`   | Порнографический контент  |
| `SCAM`          | Мошенничество             |
| `PHISHING`      | Фишинг                    |
| `LINK`          | Сообщение содержит ссылку |
| `INSULT`        | Оскорбления               |
| `HATE`          | Язык ненависти            |
| `VIOLENCE`      | Насилие или угрозы        |
| `DRUGS`         | Наркотические вещества    |
| `GAMBLING`      | Азартные игры             |
| `OTHER`         | Другая категория          |

---

## Ошибки

### 400 Bad Request

Некорректное тело запроса.

```json
{
  "error": "Field 'comment' is required"
}
```

### 500 Internal Server Error

Внутренняя ошибка сервера.

```json
{
  "error": "Internal server error"
}
```
