# Hangman API — получение слова {#hangman}

:::info Base API URL
`https://api.megoru.ru`
:::

## Конечная точка

| Метод | Endpoint    | Описание        |
|-------|-------------|-----------------|
| GET   | `/api/word` | Получение слова |

---

## Query-параметры

| Параметр   | Тип     | Обязательный | Описание                                      |
|------------|---------|--------------|-----------------------------------------------|
| `language` | String  | да           | Допустимые значения: `RU`, `EN`               |
| `category` | String  | нет          | Примеры: `colors`, `flowers`, `fruits`        |
| `length`   | Integer | нет          | Получить слово, не превышающее заданную длину |

**Примеры:**
- `?language=RU&category=fruits`
- `?language=EN`
- `?length=5`

---

## Заголовки

| Имя            | Тип    | Описание           |
|----------------|--------|--------------------|
| `Content-Type` | String | `application/json` |

## Ответы

### 200 OK
```json
{
  "word": "яблоко"
}
```

### 400 Bad Request
:::warning
Возвращает если отсутствует обязательный параметр `language`, неверное значение, или другие проблемы с валидацией.
:::

```json
{
  "error": "Api error"
}
```

:::warning
Возвращает если длина которого не превышает заданное значение
:::

```json
{
  "error": "Couldn't find a word for the specified length."
}
```

### 500 Internal Server Error
```json
{
  "error": "Internal server error"
}
```
