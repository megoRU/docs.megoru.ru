# Hangman API — получение слова {#hangman}

:::info Base API URL
`https://api.megoru.ru`
:::

---

## Endpoint

```
GET /api/word
```

---

## Query-параметры

| Параметр   | Тип    | Обязательный | Описание                                      |
|------------|--------|--------------|-----------------------------------------------|
| `language` | string | да           | Допустимые значения: `RU`, `EN`               |
| `category` | string | нет          | Примеры: `colors`, `flowers`, `fruits`        |
| `length`   | string | нет          | Получить слово, не превышающее заданную длину |

**Примеры:**
- `?language=RU&category=fruits`
- `?language=EN`

---

## Ответы

### 200 OK
```json
{
  "word": "town"
}
```
Пример на русском:
```json
{
  "word": "яблоко"
}
```

### 400 Bad Request
Причина: отсутствует обязательный параметр `language`, неверное значение, или другие проблемы с валидацией.

```json
{
  "error": "Api error"
}
```

Причина: не найдено слово, длина которого не превышает заданное значение

```json
{
  "error": "Couldn't find a word for the specified length."
}
```

### 500 Internal Server Error
```json
{
  "error": "Internal server error"
}
```

---

## Схема ответа
```json
{
    "word": "string"
}
```

---

## Замечания
- `language` — обязательный параметр. Если его нет или он неправильный — сервер возвращает 400.
- `category` — опционален; если не указан — выдаётся случайное слово из всех категорий.
- Все ответы — в `application/json`.
- Максимальная длина слова зависит от внутренних настроек сервера.
