### Поиск игры

**Отправка:**
```json
{
  "type": "FIND_GAME",
  "lang": "ru",
  "name": "string",
  "image": "base64"
}
```

**Ответы:**

Если игроков нет:
```json
{
  "type": "WAITING"
}
```

Если найден соперник, отправляется обоим:
```json
{
  "type": "MATCH_FOUND",
  "gameId": "EnVoLyvjQAaRXwVXbH",
  "wordLength": 8,
  "players": [
    { "name": "Иван", "image": "base64" },
    { "name": "Олег", "image": "base64" }
  ]
}
```

---

### Как делать ход

**Отправка:**
```json
{
  "type": "MOVE",
  "gameId": "EnVoLyvjQAaRXwVXbH",
  "letter": "а"
}
```

**Ответ:**
```json
{
  "attemptsLeft": 7,
  "duplicate": false,
  "type": "STATE_UPDATE",
  "maskedWord": "_ _ _ _ _ _ _ _"
}
```

---

### Выход из игры

**Отправка:**
```json
{
  "type": "LEAVE_GAME",
  "gameId": "AbCdEf123456"
}
```

**Ответ:**
```json
{
  "type": "GAME_CANCELED",
  "word": "город"
}
```

---

### Результат окончания игры
> Приходит при выходе игрока или после MOVE если требуется

**Ответ для соревновательной игры:**
```json
{
  "type": "GAME_OVER",
  "result": "WIN",
  "word": "город"
}
```

---

## Пример последовательности для начала игры (соревновательный режим)

1. Отправляете `FIND_GAME`.
2. Получаете `MATCH_FOUND` или `WAITING`.
3. При `MATCH_FOUND` начинаете отправлять `MOVE` и получать `STATE_UPDATE`.

---

## Пример последовательности при выходе из игры (соревновательный режим)

1. Отправляете `LEAVE_GAME`.
2. Сервер всегда вернет другому игроку `GAME_CANCELED`
