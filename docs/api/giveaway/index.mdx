## Конечная точка

| Метод | Endpoint            | Описание                 |
|-------|---------------------|--------------------------|
| POST  | `/api/participants` | Определение победителей  |

---

## Тело запроса (Request Body)

| Параметр | Тип     | Обязательно | Описание               |
|----------|---------|-------------|------------------------|
| `n`      | Integer | Да          | Количество победителей |
| `min`    | Integer | Да          | Минимальное значение   |
| `max`    | Integer | Да          | Максимальное значение  |

---

## Заголовки

| Имя            | Тип    | Описание           |
|----------------|--------|--------------------|
| `Content-Type` | String | `application/json` |

---

## Ответы сервера

### 200 OK — Успешное определение победителей

```json
{
  "winners": ["3", "1"]
}
```

### 503 Service Unavailable

```json
{
    "error": "500"
}
```
